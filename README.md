# TimerLink

# Proje Amacı

TimerLink ile süreli url linki paylaşmanıza imkan tanır, siz bir url yi oluşturur süresini belirlersiniz o süre dolduktan sonra url artık bir işe yaramaz.

# Neler Lazım

- Laravel 7.x
- Mysql
